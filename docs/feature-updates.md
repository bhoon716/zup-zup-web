# Web Release Notes

`web` 모듈의 정식 릴리스 UI/UX 명세입니다.

---

## [v1.2.0] - 2026-03-23 🚀

서비스 브랜딩을 '전북대 수강신청 도우미'에서 **'줍줍 (zup-zup)'**으로 전면 개편했습니다.

### Changed
- **UI 및 메타데이터**: 메인 타이틀, 헤더 로고, 푸터 카피라이터, 로그인 카드 및 약관의 서비스 명칭을 **'줍줍'**으로 전면 교체
- **PWA 매니페스트**: `manifest.json`의 앱 설치명을 `줍줍`으로 일치화
- **SEO 최적화**: 페이지 타이틀 구조를 `줍줍 (zup-zup)`으로 일관성 있게 구성

---

## [v1.1.1] - 2026-03-10 🛠️

서비스 전반의 용어 정제 및 관리자 기능 고도화 작업이 진행되었습니다.

### Added

- **운영진 답변 삭제**: 관리자 페이지에서 잘못 작성되거나 불필요한 답변을 영구 삭제하는 기능 및 액션 로그 연동
- **이미지 외부 접근 자동화**: 업로드된 첨부 파일을 인증 없이도 안전하게 조회할 수 있도록 보안 정책 및 프록시 레이어 구축

### Changed

- **용어 전면 개편**: "피드백(Feedback)"으로 표기되던 모든 명칭을 사용자 친화적인 **"문의 및 건의"**로 통일 (UI, API, 주석 포함)
- **어드민 인터페이스 정교화**:
  - 처리 상태(대기/처리중/완료/반려) 및 문의 유형(버그/건의/기타)을 직관적인 한국어 배지로 대치
  - 답변 작성 시 실시간 통계 반영 및 수정 모드 UI 시인성 보완
- **개인정보 노출 최소화**: 일반 문의 목록 및 상세 뷰에서 작성자의 환경 정보(OS, 브라우저)를 숨기고 관리자 페이지에서만 확인 가능하도록 UX 조정

---

## [v1.1.0] - 2026-03-09 🚀

강의 리뷰 인터페이스 고도화 및 통합 건의/제보 시스템이 도입되었습니다.

### Added

- **통합 건의/제보 시스템(Feedback System)**: 버그 리포트, 기능 제안 등을 서비스 내부에서 직접 제출할 수 있는 인터페이스 추가
- **이미지 최적화 업로드**: 브라우저 단에서 WebP 변환 및 리사이징 처리를 통해 모바일 환경에서도 부담 없는 고속 업로드 구현
- **피드백 이력 관리**: 본인이 남긴 문의 내역을 탭 형태로 즉시 확인하고, 관리자의 답변을 실시간으로 피드백 받을 수 있는 UI 구축
- **관리자 전용 대시보드 확장**: 제보된 피드백의 상태 변경(대기/처리중/완료) 및 답변 등록, 수정 기능이 포함된 대규모 관리 인터페이스 도입
- **환경 메타데이터 자동 수집**: 제보 시점의 OS, 브라우저, 발생 경로 정보를 함께 전송하여 개발팀의 정확한 재현 및 디버깅 지원
- **리뷰 섹션(Review Section)**: 강의 상세 모달 하단에 별점 및 리뷰 목록 인터페이스 추가
- **사용자 중심 정렬**: 내가 작성한 리뷰가 리스트의 가장 앞에 고정되어 즉시 확인 가능하도록 UX 개선
- **반응형 리액션 버튼**: 각 리뷰에 대해 공감/비공감 피드백을 남길 수 있는 인터랙티브 버튼 도입
- **프리미엄 알림(Sonner)**: 브라우저 기본 알럿을 지양하고 `sonner` 라이브러리를 통한 세련된 토스트 메시지 시스템 구축
- **로그인 유도 시스템**: 비인증 사용자가 리뷰 관련 액션 수행 시 로그인 모달을 즉시 호출하도록 개선

### Changed

- **내비게이션 바 메뉴 개편**: 헤더 영역에 '건의사항' 메뉴를 상시 노출하여 접근성 극대화
- **소프트 삭제 정책 도입**: 유저가 삭제한 피드백을 데이터베이스에 안전하게 보존하면서 목록에서는 즉시 제거하여 데이터 무결성 확보
- **작성 폼 노출 최적화**: 중복 리뷰 작성 시 안내 배너 대신 작성 폼을 자동 은닉 처리하여 UI 노이즈 최소화
- **강의 상세 카드 디자인**: 우측 하단에 해당 강의의 평균 별점 및 리뷰 수를 실시간 반영하도록 레이아웃 보완
- **용어 일관성 정비**: 서비스 전반의 "한줄평" 명칭을 "리뷰"로 통일
- **PWA 설치 접근성**: 비로그인 상태에서도 홈 화면 바로가기/설치 버튼이 노출되도록 정책 변경
- **데이터 동기화 고도화**: TanStack Query를 통해 리뷰 및 피드백 액션 후 정보를 지연 없이 갱신

## [v1.0.0] - 2026-03-01 ✅

줍줍 웹 플랫폼의 모든 핵심 인터페이스를 통합한 정식 릴리스입니다.

### Added

- **그리드 기반 강의 검색**: 드래그 앤 드롭으로 시간을 직접 선택하거나 공강 기간을 자동 반영하는 스마트 필터
- **실시간 무한 스크롤**: TanStack Query를 활용하여 수천 개의 강의 데이터를 끊김 없이 탐색
- **공지사항(Announcement) 페이지**: 목록(미리보기)과 상세(마크다운 렌더링) 라우트 분리 및 검색 기능
- **통합 관리자 패널**: Recharts 기반의 통계 가시화, 크롤링 타겟 제어, 학사 일정 CRUD 화면
- **시간표 시뮬레이터**: 수업 시간표 생성 및 겹치는 강의 세로 분할 렌더링 로직 구현
- **Self-Healing 웹푸시**: VAPID 키 불일치나 만료된 구독을 자동으로 탐지하고 소생시키는 로직
- **카카오맵 SDK 연동**: 강의실 위치 정보를 지도에 즉각 매핑하여 시각화
- **사용자 온보딩**: 신규 가입 시 알림 채널 설정 및 필수 정보 입력을 돕는 단계별 가이드
- **공용 마크다운 렌더러**: `MarkdownViewer` 컴포넌트를 표준 UI로 도입하여 문서 렌더링 규칙 통일

### Changed

- **관리자 공지 편집 페이지 개편**: 목록 탐색과 편집/미리보기를 4:6 분할 레이아웃으로 최적화
- **마이페이지 알림 설정**: 10초 쿨타임 및 카운트다운을 포함한 실제 동작 테스트 버튼 제공
- **반응형 디자인 보완**: 모바일 환경에서의 시간표 스케일, 폰트 및 정렬 가독성 정밀 보정
- **검색 UX 최적화**: 불필요한 검색 버튼 및 중복 인터랙션을 제거한 직관적인 필터 배치
